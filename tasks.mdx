title: "Tasks"
description: "Create and manage to-dos and reminders linked to your contacts and relationships."
---

Tasks in Orbis are to-dos and reminders you can attach to contacts, companies, or keep standalone. Use them to stay on top of follow-ups, track relationship actions, and coordinate work with your team.

## Creating a task

You can create a task from two places:

- **The Tasks view** — click **New task** from the main Tasks view
- **A contact or company profile** — click **Add task** from the Tasks tab on the profile

When creating a task, you can set:

- **Title** — a short description of what needs to be done
- **Description** — optional additional notes
- **Due date and time** — when the task should be completed
- **Status** — the task's current state
- **Linked contacts** — one or more contacts this task relates to
- **Linked companies** — one or more companies this task relates to
- **Assignees** — team members responsible for completing the task
- **Visibility** — private (only you) or organization (visible to your team)

## Task statuses

Tasks move through the following statuses:

| Status | Meaning |
|--------|---------|
| **Queued** | Waiting to be started |
| **Inbox** | Needs triage or attention |
| **In Progress** | Actively being worked on |
| **Review** | Waiting for review or approval |
| **Done** | Completed |

## Task priorities

Each task can have one of four priority levels:

| Priority | When to use |
|----------|------------|
| **Low** | Nice to do, no deadline pressure |
| **Medium** | Standard priority |
| **High** | Important, needs attention soon |
| **Urgent** | Must be done immediately |

## Marking tasks complete

To complete a task, change its status to **Done** using the status selector in the task row or the task detail drawer. You can also bulk-update statuses by selecting multiple tasks in the list.

## The Tasks view

The main Tasks view gives you a full picture of all tasks across your workspace.

### List view

The list view shows tasks in a table with columns for title, status, priority, due date, linked contacts, linked companies, and assignees. You can:

- **Filter** by status, priority, due date range, assigned contact, or assignee
- **Sort** by due date, priority, or creation date
- **Bulk select** tasks to update status or delete in bulk
- **Search** across task titles

### Kanban view

Switch to the kanban view to see tasks organized as cards across status lanes. Drag cards between lanes to update their status.

<Tip>
Use the kanban view for a quick overview of your pipeline of relationship actions, and the list view when you need to filter by due date or contact.
</Tip>

## Reminders and notifications

Orbis sends a notification when a task's due date arrives. Make sure your notification preferences are configured in **Settings** to receive task reminders.

<Note>
Tasks linked to a contact also appear on the contact's profile under the **Tasks** tab, so you can see all pending actions for a relationship in one place.
</Note>

## Assigning tasks to team members

When working in an organization, you can assign tasks to any active team member. The assignee sees the task in their own Tasks view and receives a notification when the task is due.

Tasks default to assigning to yourself when created, but you can change or add assignees at any time.

## Visibility

Tasks can be:

- **Private** — only visible to you (default)
- **Organization** — visible to all members of your organization

Set visibility when creating the task using the toggle in the task creation dialog.
